Sport types

Sport type resource management service.

Get available sport types

Returns the list of sport types.

Headers

Custom accept header must be provided when requesting a sports resource. It defines vendor name, API version and content type.

Accept: application/vnd.aracrown.v1+json

Method

GET /sports

Query parameters

Name Description
f Position of the first result, numbered from 0. Default is 0.
p Maximum number to return. Default is 10. Available maximum - 500.
name Filter sport types by name.

Result

Json object of elements is returned as a result to request.

Name Description
total Total count of query results.
count How may items are returned by this query result.
pageSize Page size for this response.
currentPage The current page.
items Collection of items returned.
Resource property Description
uniqueIdentifier (id, link) The sport type unique identifier, containing resource id and the link to this resource.
name The sport type name.

Response codes

Examples

Request

GET /sports?f=0&p=10&name=test

Response

{
   "total": 1,
   "count": 1,
   "pageSize": 10,
   "currentPage": 0,
   "items": [   {
	  "uniqueIdentifier":       {
		 "id": 10000,
		 "link": "/sports/10000"
	  },
	  "name": "TestUpdate3211556"
   }]
}

Get concrete sport type

Returns the details of selected sport type.

Headers

Custom accept header must be provided when requesting a sports resource. It defines vendor name, API version and content type.

Accept: application/vnd.aracrown.v1+json

Method

GET /sports/{id}

Query parameters

Name Description
id The unique identifier of sport type to retrieve.

Result

Json object of elements is returned as a result to request.

Name Description
uniqueIdentifier (id, link) The sport type unique identifier, containing resource id and the link to this resource.
name The sport type name.

Response codes

Examples

Request

GET /sports/10000

Response

{
   "uniqueIdentifier":    {
      "id": 10000,
      "link": "/sports/10000"
   },
   "name": "TestInit70326078"
}

Leagues of the sport type

Returns the list of leagues for the selected sport type.

Headers

Custom accept header must be provided when requesting a sports resource. It defines vendor name, API version and content type.

Accept: application/vnd.aracrown.v1+json

Method

GET /sports/{id}/leagues

Query parameters

Name Description
id The unique identifier of sport type.

Result

Json object of elements is returned as a result to request.

Name Description
total Total count of query results.
count How may items are returned by this query result.
pageSize Page size for this response.
currentPage The current page.
items Collection of items returned.
Resource property Description
uniqueIdentifier(id, link) The league unique identifier, containing resource id and the link to this resource.
name The league name.
participantType

The participant type.

Available participant types:

/** Single player type. */
INDIVIDUAL,

/** Team type. */
TEAM,

/** Other type, not covered by enumeration. */
OTHER
leagueType

The league type.

Available league types:

/** Different elimination types. */
ELIMINATION,

/** Group tournament type. */
GROUP,

/** ELIMINATION, GROUP. */
MULTI_STAGE,

/** Other type, not covered here. */
OTHER;
country(id, link) The country unique identifier of this league.
sportType(id, link) The sport type unique identifier of this league.
